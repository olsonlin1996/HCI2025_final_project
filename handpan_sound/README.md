# Handpan Samples

Place 16-bit mono/stereo WAV files matching the note names used by the interface. Suggested filenames:

- c1.wav / d1.wav / e1.wav / f1.wav / g1.wav
- a1.wav / b1.wav (optional for extended pentatonics)

Additional mapping examples:
- `gong` -> c1.wav
- `shang` -> d1.wav
- `jue` -> e1.wav
- `zhi` -> g1.wav
- `yu` -> a1.wav
